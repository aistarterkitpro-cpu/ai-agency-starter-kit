## Week 1 — Foundation

### Admin (Day 1)
- [ ] Send signed contract
- [ ] Collect 50% deposit
- [ ] Send welcome email with timeline
- [ ] Schedule kickoff call
- [ ] Add client to project management tool

### Kickoff Call (Day 2–3)
- [ ] Confirm goals and success metrics
- [ ] Collect all credentials/access needed
- [ ] Agree on communication cadence (weekly update? Slack channel?)
- [ ] Set 30-day milestone

### Access & Setup (Day 3–5)
- [ ] Get access to relevant platforms (CRM, website, phone system, etc.)
- [ ] Document current workflow (before state)
- [ ] Identify integration points
- [ ] Set up dedicated workspace/folder for this client

---

## Week 2 — Build

- [ ] Build Phase 1 of deliverable
- [ ] Internal test/QA
- [ ] Share preview with client for feedback
- [ ] Revise based on feedback
- [ ] Document what was built

---

## Week 3 — Launch

- [ ] Final QA pass
- [ ] Client approval sign-off
- [ ] Go live
- [ ] Monitor for 48 hours post-launch
- [ ] Send "We're Live" update to client

---

## Week 4 — Optimize

- [ ] Pull first performance data
- [ ] Schedule 30-day review call
- [ ] Identify 2–3 quick wins or improvements
- [ ] Collect 50% final payment (if applicable)
- [ ] Pitch retainer or next service

---

## Post-Launch (Ongoing)
- [ ] Monthly performance report
- [ ] Quarterly strategy review
- [ ] Ask for testimonial/case study at 60 days
- [ ] Ask for referral at 90 days
